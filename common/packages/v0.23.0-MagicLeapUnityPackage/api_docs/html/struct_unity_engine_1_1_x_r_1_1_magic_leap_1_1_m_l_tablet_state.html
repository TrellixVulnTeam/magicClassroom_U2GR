int [] UnityEngine.XR.MagicLeap.MLTabletState.AdditionalPenTouchData

Additional coordinate values (x, y, z) It could contain data specific to the device type.

AdditionalPenTouchData for Wacom holds pen tilt data (x, y), in degrees from -64 to 64. Straight up an down is 0.
