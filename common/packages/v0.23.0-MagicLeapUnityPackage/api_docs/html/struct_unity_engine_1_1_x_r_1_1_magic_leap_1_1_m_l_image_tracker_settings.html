uint UnityEngine.XR.MagicLeap.MLImageTrackerSettings.MaxSimultaneousTargets

Maximum number of Image Targets that can be tracked at any given time.

If the tracker is already tracking the maximum number of targets possible then it will stop searching for new targets which helps reduce the load on the CPU. For example, if you are interested in tracking a maximum of x targets simultaneously from a list y (x < y) targets then set this parameter to x.

The valid range for this parameter is from 1 through 25.
