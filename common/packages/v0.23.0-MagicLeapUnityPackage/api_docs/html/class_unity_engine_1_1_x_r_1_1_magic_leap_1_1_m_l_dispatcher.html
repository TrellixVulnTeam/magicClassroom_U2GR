Detailed Description

This interface is to let an application query the platform to handle things that the app itself cannot or wants someone else to handle. For example, if an application comes across a schema tag that it doesn't know what to do with, it can query the patform to see if some other application might. This can be useful for handling http://, https:// or other customSchema://<arg1>/<arg2> Apart from handling schema tags in URIs, this interface can also be used to query the platform to handle a type of file based on file-extension or mime-type
