Parameters for a raycast request.

Class Members
bool CollideWithUnobserved If true, a ray will terminate when encountering an unobserved area and return a surface or the ray will continue until it ends or hits a observed surface.
Vector3 Direction Direction of the ray to fire.
uint Height The number of vertical rays. For single point raycast, set this to 1.
float HorizontalFovDegrees The horizontal field of view, in degrees.
Vector3 Position Where the ray is cast from.
Vector3 UpVector Up vector of the ray to fire. Use (0, 0, 0) to use the up vector of the rig frame.
uint Width The number of horizontal rays. For single point raycast, set this to 1.
