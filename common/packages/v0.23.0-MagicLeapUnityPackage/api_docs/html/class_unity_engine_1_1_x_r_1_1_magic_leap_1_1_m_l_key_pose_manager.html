Enables or disables an array of KeyPoses.

Parameters
keyPoses
Parameters
enableEnable or disable KeyPoses.
exclusiveWhen enabling and this is true, only the list of provided KeyPoses are enabled, all other previously-enabled KeyPoses get disabled. No effect if parameter enable is false.

Enabling too many KeyPoses will currently lead to decreased sensitivity to each individual KeyPose.
